# Userscript Collection

## Introduction

Welcome to the **Userscript Collection** repository 👋  

This repository contains a collection of **userscripts** created to enhance, customize, and automate your web browsing experience. These scripts work with popular userscript managers such as **Tampermonkey**, **Greasemonkey**, and **Violentmonkey**.

The goal of this repository is to make websites more convenient, efficient, and enjoyable by adding useful features, removing limitations, and automating repetitive tasks.

## Features

- 🚀 Enhance website functionality and usability  
- 🛠️ Automate repetitive actions  
- 🎨 Customize website appearance and behavior  
- 🧩 Lightweight and easy to install  
- 🔄 Easy to maintain and extend  

## Requirements

To use these userscripts, you need to install a userscript manager in your browser:

- Tampermonkey (Chrome, Edge, Firefox)
- Violentmonkey
- Greasemonkey (Firefox)

## Installation

1. Install a userscript manager for your browser.
2. Open the userscript file from this repository.
3. Click **Install** in your userscript manager.
4. Refresh the target website to activate the script.
